Responsive Redesign

Responsive Redesign is one of the projects for my UIUX class. We were assigned the task of choosing a not so good website, analyse usability, learnability, and memorability problems, create lofi and hifi prototypes and then finally recreate the website accordingly.

Chosen Website: Petersbuss

Problems with the website

  • Usability, Learnability, and Memorability

    1. There's no clear text hierarchy. Font size is sometimes too small.

    2. Home page is cluttered with a lot of information

    3. Navigation is confusing with too many links in home page.

    4. Webpage doesn't use common terms.

    5. Webpage is not responsive

    6. There's no clear call to action.

    7. The contact information is spread out.

    8. There are too many unnecessary colors.

    9. The images on the home page are low resolution and have distracting backgrounds

  • Accessibility

    1. No defined language for screen reader.

    2. Missing alt text for images due to which screen reader cannot interpret correctly.

    3. Very low contrast

    4. No heading structure and page regions to provide navigation functionality to assistive technology users
