Development in React.js

I made a shopping website as my first project in React to understand front-end engineering!

Goals and Value of the application

  1. Many shoppers have a go-to shop to buy their clothes. What if they could find all their brands in one place? Better yet, it would be great to filter by brands.

  2. Another common filter many shopping websites have proven to be useful and for users to be helpful is the option to filter by color.

  3. The most common sorting option users want is the price. Let's face it, many of us are broke.

  4. Looking at the shopping cart itself, users find it nice to have a remove button for every element instead clearing the cart for a single change.

Usability Principles Considered

  1. User control and freedom by providing different options for sorting and filtering.

  2. Consistency of wording with common knowledge vocabulary.

  3. Ability to get back to original state without refreshing page.

  4. Accessibility by providing alt descriptions with helpful messages for screen reader.
