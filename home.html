A/B Testing

I was in the process of deisgning a cactus shop when I couldn't decide between a desert background or a plain white background, and shopping cart in the bottom or top. So I decided to do some A/B Testing.

Version A and Version B

I created two versions of the same cactus shop. In the link above, I altered the code so that Version A would be displayed half the time and Version B would be displayed the other half.

Hypothesis for testing

In order to decide which html background and shopping cart position worked better, I used two metrics: Time Completion for the user to complete the task and the Return Rate of the users after checkout. I started by making null and alternate hypotheses for each of them.

Task: Add $150 worth of cacti to the cart and checkout!

Time completion

Null Hypothesis: Version A will have an equal or lower time completion rate than Version B

Alternate Hypothesis: Version B will have a higher time completion rate than Version A

Reasoning: The two different features of Version B is that images on Version B change on hover and the cart is placed at the top of the page. The image hover would encourage the user to hover over each image first and then add to cart, increasing the time. The position of the cart forces the user to scroll up to check the contents of the cart, thereby increasing time.

Return Rate

Null Hypothesis: Version A will have an equal or lower return rate than Version B

Alternate Hypothesis: Version B will have a higher return rate than Version A

Reasoning: Version B is more aesthetically pleasing with a neutral combination, better quality images and more features (image hover). This would encourage users to return to the site to explore further.

Data Collection

I collected data from 40 different users with the same task as above to determine the result precisely. Half of the data collected was from peers in my studio. The other half consisted of my friends and family who did not have a background on this project and only knew the task assigned to them. This could potentially reduce bias.
