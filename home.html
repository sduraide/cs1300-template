Hyperlink

Introduction

Hyperlinks are the essence of web. For a user experience, designers need to create a hyperlink structure that allows various kinds of browsing content. Based on how the contents are parsed and connected to each other via hyperlinks, a designer can create a new reading of the content. In this project, I explored how we can use hyperlinks to catalogue items in a creative way.

Lofi Prototyping with Wireframes

I started off by brainstorming how I wanted my layout to look like in the form of wireframes.

The following website presents almost everything in one long page divided by sections. While scrolling the section would be fixed and the users can skip sections by clicking on the header link if they want to. They can click on images to show the text about it.

The below series of wireframes describe a two column layout where the first column would have a stationary highlighting Pierre Marcolini with the chocolate. The second column is where all the text would go. There are four sections: About, History, Manufacturing and Gallery.

Conclusion
